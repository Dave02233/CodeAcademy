# Media vs Mediana - Differenza FACILE

## MEDIA

**"Somma tutto e dividi"**

**Formula**: (1+2+3+4+100) ÷ 5 = 22

**Problema**: Un riccone (100) tira su tutto!

## MEDIANA

**"Ordina e prendi il CENTRO"**

**Formula**: Ordina [1,2,3,4,100] → Prendi 3° = 3

**Vantaggio**: Ignora ricconi e poveri estremi!

## Esempio Stipendi

| Dati                         | Media  | Mediana | Vincitore   |
| ---------------------------- | ------ | ------- | ----------- |
| [1000,1200,1300,1400,1500]   | 1280€  | 1300€   | Pareggio    |
| [1000,1100,1200,1300,100000] | 20826€ | 1200€   | **MEDIANA** |

## Quando USARE Cosa

| Situazione       | Usa MEDIA | Usa MEDIANA |
| ---------------- | --------- | ----------- |
| Numeri normali   | ✅        | ✅          |
| **Con outlier**  | ❌        | ✅          |
| Stipendi, prezzi | ❌        | ✅          |
| Tempi, età       | ✅        | ✅          |
